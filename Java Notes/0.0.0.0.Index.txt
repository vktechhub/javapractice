Overview:- 
    Aiming to learn Core Java concepts and features; then advanced Java concepts. 
    Then learn Spring Framework and SpringBoot. 
    Finally, learn Microservices architecture using SpringBoot and Spring Cloud.

    This document provides a summary of key topics and concepts in each area.

* Spring Annotations
        * Core and Context Management Annotations - @Configuration, @Component, @Bean
            * Specialization of @Component - @Service, @Repository, @Controller
            * variants of @Configuration - @ComponentScan, @Import, @ImportResource
                * Conditional Annotations - @Conditional, @ConditionalOnProperty, @ConditionalOnMissingBean
        * variants of Dependency Injection - @Autowired, @Qualifier, @Value
        * Lifecycle - @PostConstruct, @PreDestroy
        * Web and Rest Annotations - @RestController, @ResponseBody, @RequestMapping, @PathVariable, @RequestParam
        * Aspect-Oriented Programming - @Aspect, @Before, @After, @Around, @Pointcut, @AfterReturning, @AfterThrowing
        * Data Access Annotations - @Transactional, @Repository, @Entity, @Table, @Id, @GeneratedValue, @Column
        * Scheduling and Asynchronous Processing - @EnableScheduling, @Scheduled, @Async
        * Caching Annotations - @EnableCaching, @Cacheable, @CachePut, @CacheEvict

    Other notes:-
        * @Controller vs @RestController
        * @Component vs @Service vs @Repository
        * @Configuration vs @Component
        * @Bean vs @Component
        * @Autowired vs @Inject vs @Resource
        * @RequestMapping vs @GetMapping vs @PostMapping vs @PutMapping vs @DeleteMapping
        * @PathVariable vs @RequestParam vs @RequestBody vs @ModelAttribute
        * @PathVariable vs @RequestParam
        * @Transactional propagation and isolation levels
