String in Go:-
    String in Go is a sequence of bytes. 
    It is immutable, meaning that once a string is created, it cannot be changed. 
    Strings in Go can contain any Unicode characters.
    Go supports two styles of string literals, the double-quote style (or interpreted literals) and the back-quote style (or raw string literals).
    You can concatenate strings using the + operator and += operators, and you can also use the fmt package to format strings.
    The zero values of string types are blank strings, which can be represented with "" or `` in literal.
    String types are all comparable (by using the == and != operators). And like integer and floating-point values, two values of the same string type can also be compared with >, <, >= and <= operators. When comparing two strings, their underlying bytes will be compared, one byte by one byte. If one string is a prefix of the other one and the other one is longer, then the other one will be viewed as the larger one.
    
    Example:-
        const World = "world"
	    var hello = "hello" 

        fmt.Println(hello + " " + World) // Output: hello world

    The Internal Structure of String Types
        In Go, a string is represented as a struct that contains a pointer to the underlying byte array and the length of the string.
                type _string struct {
                    elements *byte // underlying bytes
                    len      int   // number of bytes
                }
        * This means that when you create a string, it is stored in memory as a sequence of bytes, and the string variable holds a reference to that memory location along with the length of the string.
        * This design allows for efficient memory usage and fast string operations, as the string variable itself does not need to store the entire string data, but rather just a reference to it. 
        * When you perform operations on strings, such as concatenation or slicing, Go creates new string values without modifying the original string, ensuring immutability.
