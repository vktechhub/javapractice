1. create a new maven project using ide from org.apache.maven.archetypes - quickstart
2. update java version to 17 in pom.xml under properties
    <maven.compiler.source>17</maven.compiler.source>
    <maven.compiler.target>17</maven.compiler.target>
3. add spring context under dependencies
        <dependency>
            <groupId>org.springframework</groupId>
            <artifactId>spring-context</artifactId>
            <version>6.2.12</version>
        </dependency>
4. Create a POJO class - Plain old java objects
5. create a  Spring configuration file - XML based configuration (must be within the package of POJO)
6. update xml config from spring bean documentation
        https://docs.spring.io/spring-framework/reference/core/beans/basics.html

7.then create ApplicationContext and load the XML configuration file
8. Retrieve the bean from the context and use it
