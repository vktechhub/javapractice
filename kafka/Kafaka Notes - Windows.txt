Kafaka related notes:-

Out of 3 versions in download portal "kafka_2.13-4.1.1" is some what working.

To make that work, 
	* you need move it to personal folder, so that you don't need admin access
	* 'wmic' is not recognized as an internal or external command,
		* Open bin\windows\kafka-server-start.bat in a text editor.
		* Search for any line containing wmic.
		* Comment out those lines by adding rem at the beginning of the line.
		* Add a manual heap setting right below the commented lines:
			set KAFKA_HEAP_OPTS=-Xmx1G -Xms1G
